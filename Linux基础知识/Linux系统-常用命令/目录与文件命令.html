阿东的知识库

1. cp - 复制文件/目录

  • 核心功能：
    • 复制文件或目录到指定位置；
    • 可实现单个文件复制、多个文件批量复制、目录及内容复制。
  • 关键选项：
    • -i（--interactive）：复制前若目标文件已存在，提示确认是否覆盖；
    • -r（--recursive）：递归复制目录（必须使用此选项才能复制目录及其中内容）；
    • -v（--verbose）：显示复制过程的详细信息（如“'file.txt' -> 'dir/file.txt'”）；
    • -a（--archive）：归档复制，保留文件权限、所有者、时间戳等所有属性（常用于备份）；
    • -u（--update）：仅复制源文件比目标文件新，或目标文件不存在的情况；
    • -p：保留源文件的权限、所有者和时间戳（不包括链接等特殊属性）。
  • 常用示例：
    • cp file.txt dir/：将file.txt复制到dir目录下；
    • cp file1.txt file2.txt dir/：将多个文件批量复制到dir目录；
    • cp -r src_dir/ dest_dir/：复制src_dir目录及所有内容到dest_dir；
    • cp -i old.txt new.txt：若new.txt存在，会提示“是否覆盖”。

2. rm - 删除文件/目录

  • 核心功能：
    • 删除指定的文件或目录，操作不可逆，执行前需仔细确认目标。
  • 关键选项：
    • -r（--recursive）：递归删除目录及其下所有文件、子目录（删除目录必须加此选项）；
    • -f（--force）：强制删除，不提示确认（即使文件为只读权限，或目标不存在也不报错）；
    • -i（--interactive）：交互式删除，删除每个文件/目录前都会提示确认（输入y确认，n取消）；
    • -v（--verbose）：显示删除过程的详细信息（如“removed 'test.txt'”）；
    • 常用组合：rm -rf 目录名（强制删除目录及内容，高危操作）、rm -i *.log（交互式删除所有.log文件）。
  • 风险提示：
    • 绝对禁止在根目录（/）执行rm -rf *或rm -rf /，会直接删除系统文件导致崩溃；
    • 删除隐藏文件需明确指定，如rm -f .bash_history（删除历史记录文件）。

3. mv - 移动/重命名文件/目录

  • 核心功能：
    • 移动文件或目录到指定位置；
    • 对文件或目录进行重命名（同一目录下移动即重命名）。
  • 关键选项：
    • -i（--interactive）：若目标文件已存在，提示确认是否覆盖；
    • -v（--verbose）：显示移动/重命名过程（如“'old.txt' -> 'new.txt'”）；
    • -u（--update）：仅当源文件比目标文件新，或目标文件不存在时才执行移动；
    • -f（--force）：强制移动，覆盖已存在的目标文件且不提示。
  • 常用示例：
    • mv file.txt dir/：将file.txt移动到dir目录下；
    • mv oldname.txt newname.txt：将文件重命名（同一目录下）；
    • mv src_dir/ dest_dir/：若dest_dir存在，将src_dir移动到dest_dir内；若不存在，则重命名src_dir为dest_dir；
    • mv file1.txt file2.txt docs/：将多个文件批量移动到docs目录。
  • 补充说明：
    • 移动操作不会改变文件内容和属性，仅修改文件路径；
    • 跨文件系统移动时，实际是先复制再删除源文件（耗时较长）。
