阿东的知识库

1. ls - 列出目录内容

  • 核心功能：
    • 显示指定目录下的文件和子目录名称；
    • 默认显示当前工作目录的内容，且不包含以“.”开头的隐藏文件。
  • 关键选项：
    • -l：以长格式显示（包含权限、所有者、大小、修改时间等详细信息）；
    • -a：显示所有文件（包括隐藏文件，即文件名以“.”开头的文件）；
    • -h：与 -l 配合使用，将文件大小以人性化单位显示（如 KB、MB、GB）；
    • -d：仅显示目录本身的信息，而非目录下的内容（如 ls -ld /home）；
    • -R：递归显示目录及其所有子目录下的内容。
  • 补充说明：
    • 可指定目标目录，如 ls /etc 显示 /etc 目录下的内容；
    • 支持通配符，如 ls *.txt 显示当前目录下所有 .txt 后缀的文件。

2. cd - 切换工作目录

  • 核心功能：
    • 更改当前用户的工作目录，是导航Linux文件系统的核心命令。
  • 关键选项与参数：
    • 无参数：切换到当前用户的家目录（等价于 cd ~）；
    • ~：代表当前用户的家目录（如 cd ~/Documents 进入家目录下的 Documents 文件夹）；
    • .：代表当前目录（一般用于命令的路径参数，如 cd . 无实际效果）；
    • ..：代表上一级目录（如 cd .. 从 /home/user 切换到 /home）；
    • -：切换到上一次所在的工作目录（如从 /home 切换到 /etc 后，cd - 回到 /home）；
    • 绝对路径：从根目录（/）开始的完整路径，如 cd /var/log；
    • 相对路径：从当前目录开始的路径，如当前在 /home，cd user/Downloads 等价于 cd /home/user/Downloads。
  • 补充说明：
    • 使用 pwd 命令可查看当前所在的工作目录，常与 cd 配合使用；
    • 目录名称中包含空格时，需用引号包裹或转义，如 cd "My Documents" 或 cd My\ Documents。
